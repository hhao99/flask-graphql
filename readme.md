Flask GraphQL App

fork from https://github.com/dev4hobby/flask-skeleton

* FLask > 2.0
* Flask-SQLAlchemy
* Redis
* OAuth


# quick start
git clone git://github.com/hhao99/flask-graphql.git

cd flask-graphql
python3 -m venv .env
source .env/bin/activate
pip install -r requirements

## develop
flask run --with-threads --reload
